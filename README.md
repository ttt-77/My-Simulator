# My-Simulator
## Running Step
### Git Clone  
```
git clone https://github.com/ttt-77/My-Simulator.git 
cd My-Simulator
```    
### Set up conda env
```
conda env create --file env.yaml 
conda activate My-Simulator
```
### Run app
```
python app.py
```
### Access on browser
```
localhost:5000
```
